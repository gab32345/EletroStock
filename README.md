EletroStock - Sistema de Controle de Estoque
Status do projeto: Em desenvolvimento (Conclusão da Etapa 5).

Tecnologias Aplicadas
Linguagem: Java.

Interface: Java Swing (Desktop).

Banco de Dados: MySQL.

Conectividade: JDBC.

Versionamento: Git e GitHub.

Time de Desenvolvedores
Nome: Gabriel.

Objetivo do Software
O EletroStock é um sistema desenvolvido para gerenciar o inventário de uma loja de eletrodomésticos. O objetivo principal é permitir o controle de produtos de forma digital, substituindo métodos manuais por um banco de dados seguro e organizado.

Funcionalidades (Requisitos)
Cadastro de Produtos: Permite inserir novos itens com nome, preço e quantidade.

Listagem Dinâmica: Exibe todos os produtos cadastrados em uma tabela integrada ao MySQL.

Busca/Filtro: Localiza produtos específicos através do campo de pesquisa por nome.

Exclusão: Permite remover produtos do estoque diretamente pela interface.
